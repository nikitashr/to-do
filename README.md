to-do
=====

The structure of the URLS

    a. root
    Lists all the todos in the system
    
    b. /tasks/new
    For to create new todo
    
    c. /tasks/1
    See the todo 1
    
    d. /tasks/1/{edit, destroy}
    Edit or delete the todo
    
    e. /tasks/1/finish
    Mark todo as finished
    
    f. /tasks/1/undo_finish
    Mark todo as unfinished
